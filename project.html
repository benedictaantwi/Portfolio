GIS Projects

Exploring spatial data through GIS-based mapping, land use classification, and suitability analysis.

Suitability and Overlay Analysis for Marine Sanctuaries

  • Applied GIS overlay techniques to analyze and rank marine habitats.

Cost Path Analysis for Grizzly Bears

This GIS project aimed to determine the least-cost path for grizzly bears moving across a fragmented landscape, ensuring optimal corridors for wildlife movement.

  • Utilized cost-distance analysis to evaluate terrain difficulty and habitat barriers.
  • Incorporated road networks and land cover classifications into the model.
  • Generated least-cost paths to identify viable wildlife corridors.

Spatial Interpolation and Visualization of LiDAR

This project involved creating Digital Elevation Models (DEMs) using various interpolation techniques on LiDAR data from the UBC Malcolm Knapp Research Forest.

  • Compared **Natural Neighbor, Spline, and IDW interpolation** methods.
  • Analyzed spatial accuracy by comparing interpolated DEMs with reference datasets.
  • Identified Natural Neighbor as the most stable interpolation method.

Terrain Analysis and Riparian Area Management

This project focused on defining reserve (no-harvest) areas in a riparian forest using GIS-based hydrological modeling. Using a Digital Elevation Model (DEM), stream classification, and buffer analysis, the project aimed to balance ecological conservation with forest resource management.

  • Modeled stream networks using **flow direction and accumulation analysis**.
  • Defined **Riparian Management Areas (RMAs)** based on stream classification.
  • Calculated buffer zones for no-harvest areas to protect sensitive watersheds.
  • Assessed the effectiveness of the buffer zones in **preventing runoff impacts on streams**.

Remote Sensing Projects

Exploring satellite imagery, LiDAR data, and environmental monitoring through advanced remote sensing techniques.

R for Remote Sensing Data Analysis

This project focused on using R programming for handling and analyzing spatial data, covering vector and raster spatial data processing in GIS and remote sensing applications.

  • Indexing spatial data for extraction, transformation, and querying.
  • Manipulating geospatial datasets using the **sf package**.
  • Visualizing spatial data with **ggplot2** for thematic maps and overlays.
  • Writing custom functions to automate geospatial analysis.
  • Processing temperature and humidity datasets for environmental insights.

The EMS and Basic Image Analysis in ENVI

This project explored the Electromagnetic Spectrum (EMS) and its applications in remote sensing using ENVI 6.0 software. It focused on spectral properties, image processing techniques, and visualization of different spectral bands.

  • Analyzed spectral reflectance and how different materials interact with electromagnetic waves.
  • Created and interpreted **true and false color composites** for various applications.
  • Worked with **panchromatic and high-resolution imagery** for detailed analysis.
  • Performed **image enhancement and histogram analysis** to improve data interpretation.

Lidar-Based Forest Attribute Estimation

This project utilized lidar point cloud data from the Alex Fraser Research Forest to estimate key forest attributes such as Above Ground Biomass (AGB) and Dominant Tree Height.

  • Developed **Digital Elevation Model (DEM)** and **Canopy Height Model (CHM)** at 2m resolution.
  • Extracted **point cloud metrics** from lidar tiles for plot locations.
  • Built predictive models for **forest biomass and tree height** using **forward variable selection**.
  • Created **scatter plots** comparing predicted vs measured values for model validation.

Measuring Spectra with the ASD FieldSpecPro

This project focused on field spectral measurements using the ASD FieldSpecPro spectrometer to analyze the spectral properties of various materials and simulate mixed pixel interactions.

  • Collected **broadleaf, conifer, dead vegetation, soil, and rock** samples for spectral analysis.
  • Used **ViewSpecPro software** to calibrate and average spectral readings.
  • Simulated **mixed pixels** to understand spectral interactions in satellite imagery.
  • Calculated **NDVI and NDWI** to analyze vegetation health and water content.
